# ☀️ EcoHelioPulse

**Профессиональный калькулятор и симулятор солнечных энергосистем**

> Проект создан для хакатона. Полный расчёт солнечных систем: от физики панелей до финансового анализа и эко-оценки.

---

## 🚀 Быстрый старт (без установки)

Просто откройте файл **`EcoHelioPulse.html`** в любом современном браузере.  
Никаких серверов, Node.js или зависимостей — всё встроено в один файл.

---

## ✨ Возможности

- 🗺️ **Интерактивная ГИС-карта** — выбор локации, автоматическое получение климатических данных (Open-Meteo API)
- ⚡ **Физический движок расчётов** — DNI/DHI, температурные модели, бифациальные панели, MPPT
- 🔋 **Анализ аккумуляторов** — LiFePO4, LTO, расчёт автономности
- 💰 **Финансовый анализ** — ROI, LCOE, срок окупаемости, тарифы feed-in
- 🌱 **Эко-оценка** — расчёт CO₂, компенсация деревьями, ESG-рейтинг
- 🔒 **Электробезопасность** — диоды, SPD, RCD, заземление, кабели
- 🌍 **Многоязычный интерфейс** — Русский / English / Español

## 🛠️ Разработка (для разработчиков)

### Требования
- Node.js 18+
- npm

### Установка и запуск
```bash
# Клонировать репозиторий
git clone https://github.com/ВАШ_ЛОГИН/ecoheliopulse.git
cd ecoheliopulse

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

### Сборка production-версии
```bash
npm run build

# Сгенерировать standalone HTML файл
python scratch/build_standalone.py
```

## 📁 Структура проекта

```

├── EcoHelioPulse.html      # ← Standalone файл (всё в одном, готов к запуску)

│   ├── favicon.svg
│   └── icons.svg
└── scratch/
    └── build_standalone.py # Скрипт генерации standalone HTML
```
